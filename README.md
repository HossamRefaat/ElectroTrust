**ElectroTrust** is a full-featured e-commerce platform for buying and selling electronic devices. Built using **ASP.NET Core MVC**, it follows a clean **Model-View-Controller** architecture to ensure scalability and maintainability.

---

## 🚀 Features

✅ User Authentication & Role-Based Access (Admin, Seller, Customer)  
✅ Sellers Can List and Manage Products  
✅ Customers Can Browse, Purchase, and Customize Products  
✅ Shopping Cart, Wishlist, and Order Management  
✅ Secure Payment Processing (PayPal, Stripe, COD)  
✅ Product Reviews & Ratings  

---

## 🏛️ Architecture & Technologies Used

- **Architecture:** ASP.NET Core MVC (Model-View-Controller)  
- **Backend:** .NET 9 (ASP.NET Core, Entity Framework Core)  
- **Frontend:** Razor Views
- **Database:** SQL Server (Using Entity Framework Core - Code First)  
- **Authentication:** Identity with JWT Authentication  
- **ORM:** Entity Framework Core  
- **Payments:** PayPal, Stripe  
- **Version Control:** Git & GitHub
